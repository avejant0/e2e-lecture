## Requirements:
- Node 14+

## How to run:
### Application:
- open Terminal
- `cd api`
- `yarn`
- `npm run start`
- open another Terminal tab
- `cd `client`
- `yarn`
- `npm run start`

Default URLS:
- API - `http://localhost:5000`
- Client - `http://localhost:3000`

### E2E
- Open terminal
- go to any folder with `e2e` prefix
- run `yarn`
For Selenium:
- `npm run test`
For Cypress:
- `npm run cypress-open`
